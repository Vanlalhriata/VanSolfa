VanSolfa is a font designed for tonic solfa notations. It is based on the Verily Serif Mono font by Bitstream Vera.

The font contains notations for three octave levels for the following:
d r m f s l t
(semitones:)
de re fe se le
ra ma sa la ta

Refer to the 'Mappings' tab in the file 'Notes-keyboard mapping.xls' to see which notes are mapped to which keys.

Please send your feedback and suggestions to vanlalhriata@gmail.com. Please note that this is a work in progress.
